# 🛒 Cartify – E-Commerce Product Browser

## 📌 Project Overview

Cartify is a web-based e-commerce product browser that allows users to explore products, search for items, and filter/sort results dynamically. The goal of this project is to demonstrate core JavaScript concepts such as API integration, array higher-order functions, and responsive UI development.

---

## 🎯 Purpose

The purpose of Cartify is to provide a simple and interactive platform where users can browse products efficiently using modern web development techniques.

---

## 📡 API Used

* **DummyJSON API**
  https://dummyjson.com/products

This API provides mock product data including title, price, category, description, and images.

---

## ✨ Planned Features

* 🔍 **Search** products by name
* 🎯 **Filter** products by category
* 📊 **Sort** products (price, alphabetical, etc.)
* ❤️ **Add to Favorites** (using localStorage)
* 🌙 **Dark/Light Mode Toggle**
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML
* CSS (or Tailwind/Bootstrap)
* JavaScript (ES6+)
* Fetch API

## 📁 Project Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/ashwinalwa30/Cartify.git
   ```
2. Navigate to the project folder:

   ```bash
   cd Cartify
   ```
3. Open `index.html` in your browser
